It's customizable and adjustable

beyondsocial.js you can use it for various social networks, and on any of your pages. Style it like you want to, there are no boundaries.

Browser Support

This plugin was was tested and works on Chrome, Safari, IE and Firefox.

Kompatibility

Beyondsocial currently works for Facebook, Tumblr, Pinterest and Instagram

Foolproof Implementation

$('#content').beyondsocial();

Load more than one feed

   $('#content').beyondsocial({
		networks: [{name: 'tumblr', id: 'beyond-t-s'}, ...], 
		maxResults: 5
	});

JavaScript

window.feedarray = new Array();
window.switcher = 0;
$('#subbutton').click(function(e) {
    if(window.switcher == 1) {
	$('#feedDisplay').beyondsocial({
	    networks: feedarray,
	});
	$('#subbutton').attr("disabled","disabled");
	$('#arraydisplay').text("Reload the page to try other feeds");
    }
});

$('#addto').click(function(e) {
    var exid = $('#exampleid').val();
    var exfeed = $('#examplefeed').val();
    feedarray.push({name: exfeed, id: exid});
    $('#arraydisplay').append(exfeed+": "+exid+"; ");
    window.switcher = 1;
});
